# academic-relational-model

# Sistema Acadêmico — Exercicio de fixação_ Database (Banco de Dados).

Este repositório contém um script SQL desenvolvido como atividade acadêmica. O objetivo foi praticar a criação de um banco de dados relacional representando um sistema simples de gestão acadêmica.


Foram criadas tabelas como pessoa, curso, aluno, professor, disciplina, turma, matrícula, provas e projetos, aplicando conceitos fundamentais de banco de dados:
Uso de chaves primárias e estrangeiras

Criação de relacionamentos entre tabelas
Aplicação de restrições como NOT NULL, CHECK e UNIQUE
Trabalhar com chaves compostas em algumas estruturas
Também foram adicionados alguns registros iniciais e algumas consultas SQL básicas apenas para verificar o funcionamento das tabelas.

Este projeto foi desenvolvido exclusivamente para fins de estudo e prática na disciplina de Banco de Dados.
